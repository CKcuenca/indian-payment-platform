1. 前言概述

概要

本文档定义了接入CashGit印度支付系统的统一API接口规范，具体包括：

  • 统一支付接口：一个API支持标准支付和唤醒支付两种方式
  • 接口通信方式：涵盖网络通讯方式及接口加解密方式
  • 业务处理流程：涉及订单创建、状态查询、手动验证等功能
  • 交易报文格式说明：包含入参、出参规范及JSON格式标准

目标人群

目标读者为具备开发能力、熟悉相关编程语言的开发、维护及管理人员。

💡 请求头说明：请求头Content-Type为：application/json;charset=utf-8，所有接口一致

相关术语

  • 平台或系统：指提供印度支付功能的CashGit系统，支持多种支付方式为商户提供完整的支付解决方案。
  • 商户号(appid)：这是商户在平台上的唯一标识，用于验证其在平台的合法性。
  • 商户订单号(orderid)：商户系统为每笔订单生成的唯一编号，需确保其唯一性。
  • 加密密钥(secretKey)：商户用于交易请求前对数据进行加密和验签的关键工具。
  • 支付类型标识(pay_id)：用于区分不同支付方式的参数，"1"表示标准支付，"2"表示唤醒支付。
  • 币种(currency)：固定为INR（印度卢比）。
  • 后台异步回调(notify_url)：当交易处理完成后，系统会主动通知商户网站并携带相关结果信息。
注意：密钥必须妥善保管，一旦泄露应及时联系平台处理，以避免可能的损失。

2. 签名算法

签名规则

Step 1. 参数排序

将非空参数值按照参数名ASCII码从小到大排序（字典序）。

  • 参数名ASCII码从小到大排序
  • 如果参数的值为空不参与签名
  • 参数名区分大小写
  • 排除sign参数本身

Step 2. 构建签名源串

使用URL键值对的格式（即key1=value1&key2=value2…）拼接成字符串。

Step 3. 拼接密钥加密

将签名源串最后拼接上secretKey，并对整个字符串进行MD5加密，得到32位小写签名值。

JavaScript签名示例

const crypto = require('crypto');

function generateSign(params, secretKey) {
  // 1. 排除sign参数并过滤空值
  const filteredParams = {};
  Object.keys(params).forEach(key => {
    if (key !== 'sign' && params[key] !== '' && params[key] !== null && params[key] !== undefined) {
      filteredParams[key] = params[key];
    }
  });
  
  // 2. 按ASCII排序并拼接
  const signString = Object.keys(filteredParams)
    .sort()
    .map(key => `${key}=${filteredParams[key]}`)
    .join('&') + secretKey;
  
  // 3. MD5加密转小写
  return crypto.createHash('md5').update(signString).digest('hex').toLowerCase();
}
重要提醒：签名验证失败的请求将被系统拒绝，请确保签名算法实现正确。

3. 统一支付接口

🔥 最新更新：现在支持统一支付接口！一个接口支持标准支付和唤醒支付两种模式。

🚀 推荐接口（统一支付接口）

POST
/api/pay

通过 pay_id 参数控制支付类型：pay_id=2 为唤醒支付，缺省或 pay_id=1 为标准支付。

🔄 兼容接口（唤醒专用）

POST
/api/wakeup/create

仅支持唤醒支付模式，向后兼容保留。新商户建议直接使用统一接口。

🎯 统一支付接口说明

统一支付接口根据 pay_id 参数自动路由到不同支付方式：

  • 标准支付方式：pay_id 缺省或等于 1
  • 唤醒支付方式：pay_id 等于 2（需要 customer_phone 参数）

商户响应

重要：商户必须验证通知的签名，确保通知来源的真实性和数据完整性。

Python示例

import hashlib
import requests
import json
import time

def generate_sign(params, secret_key):
    # 排除sign参数并过滤空值
    filtered_params = {k: v for k, v in params.items() 
                      if k != 'sign' and v not in ['', None]}
    
    # 按键名排序
    sorted_params = sorted(filtered_params.items())
    
    # 构建签名源串
    sign_string = '&'.join([f'{k}={v}' for k, v in sorted_params]) + secret_key
    
    # MD5加密
    return hashlib.md5(sign_string.encode('utf-8')).hexdigest().lower()

# 唤醒支付订单创建示例
params = {
    'appid': 'MERCHANT_MEWZV8HV',
    'orderid': 'ORDER_' + str(int(time.time())),
    'amount': '100.00',
    'desc': '游戏充值',
    'pay_id': '2',                    # 唤醒支付
    'customer_phone': '9876543210',
    'notify_url': 'https://your-game.com/notify'
}

params['sign'] = generate_sign(params, 'your_secret_key')

# 调用统一支付接口

import java.security.MessageDigest;
import java.util.*;

public class WakeupPaymentExample {
    
    public static String generateSign(Map params, String secretKey) {
        try {
            // 排除sign参数并过滤空值
            Map filteredParams = new TreeMap<>();
            for (Map.Entry entry : params.entrySet()) {
                if (!"sign".equals(entry.getKey()) && 
                    entry.getValue() != null && 
                    !"".equals(entry.getValue().toString())) {
                    filteredParams.put(entry.getKey(), entry.getValue());
                }
            }
            
            // 构建签名源串
            StringBuilder signString = new StringBuilder();
            for (Map.Entry entry : filteredParams.entrySet()) {
                signString.append(entry.getKey())
                         .append("=")
                         .append(entry.getValue())
                         .append("&");
            }
            signString.deleteCharAt(signString.length() - 1); // 删除最后的&
            signString.append(secretKey);
            
            // MD5加密
            MessageDigest md = MessageDigest.getInstance("MD5");
            byte[] digest = md.digest(signString.toString().getBytes("UTF-8"));
            StringBuilder result = new StringBuilder();
            for (byte b : digest) {
                result.append(String.format("%02x", b));
            }
            return result.toString();
            
        } catch (Exception e) {
            throw new RuntimeException("签名生成失败", e);
        }
    }
    
    // 使用示例 - 唤醒支付订单创建
    public static void main(String[] args) {
        Map params = new HashMap<>();
        params.put("appid", "MERCHANT_MEWZV8HV");
        params.put("orderid", "ORDER_" + System.currentTimeMillis());
        params.put("amount", "100.00");
        params.put("desc", "游戏充值");
        params.put("pay_id", "2");                      // 唤醒支付
        params.put("customer_phone", "9876543210");
        params.put("notify_url", "https://your-game.com/notify");
        
        String sign = generateSign(params, "your_secret_key");
        params.put("sign", sign);
        
        // 发送HTTP请求到统一支付接口 /api/pay
        // 实现HTTP POST请求逻辑...
    }
}
